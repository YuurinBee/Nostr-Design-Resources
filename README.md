# Nostr Design Resources
## Permissionless Design Library

- Free & Open-Source 
- Nostr-based Design Assets

> Greetings and welcome. <br>
> <br>
> I am a designer who often needs to find certain resources for projects and as you can imagine it can be quite difficult to find the exact assets. So, I've started this open-source design library that anyone can easily find resources, use and modify them however you want, as well as make requests. I don't own these assets, as many of them are for other projects, but simply a creative conduit helping to make it more accessible. I will update this `README` with time, but you gotta start somewhere.
